SONA Platform Elite

A premium Next.js build for the Sargodhian Overseas Network Association.

Pages included
- Homepage
- Join / Login
- Members Directory
- Global Chapters
- Founders Circle
- Announcements
- Giving

Deploy
1. Upload the repository contents to GitHub.
2. Import the GitHub repository into Vercel.
3. Deploy as a Next.js project.

Required root structure
app
components
public
package.json
next.config.mjs

All included images are already placed in public/images.
